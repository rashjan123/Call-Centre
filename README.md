# 📞 Call Center Dashboard

## 📌 Project Overview
This project presents an interactive Call Centre Analytics Dashboard built in Power BI to monitor support performance, customer satisfaction, and call handling efficiency. The dashboard helps identify trends in call volume, agent performance, and response time to improve customer service operations.

## 🎯 Project Objective

- Analyze call center performance using an interactive Power BI dashboard.
- Monitor key metrics such as total calls, response time, and customer satisfaction.
- Evaluate agent performance and identify areas for operational improvement.
- Identify trends in call volume and call handling efficiency.
- Support data-driven decision making to improve customer service operations.

## 📈 Key KPIs
- Total Calls
- Average Satisfaction
- Average Speed of Answer
- Agent Performance

## 📊 Dashboard Features
- Interactive Power BI dashboard to analyze call center performance.
- Dynamic filtering using slicers for Agent, Topic, and Date.
- Drill Down and Drill Through functionality for detailed analysis.
- Visualization of key metrics such as Total Calls, Average Satisfaction, and Response Time.
- Agent performance analysis using Matrix and comparison visuals.
- Time-based analysis of call trends using hourly and daily charts.
- User-friendly layout with tooltips and filters for better insights.

## 🔍 Key Insights
- Identified peak call hours to understand high workload periods.
- Compared agent performance based on average talk duration and customer satisfaction.
- Analyzed response time trends to evaluate call handling efficiency.
- Detected common call topics to understand major customer issues.
- Evaluated overall call center performance using key metrics and trends.

## ✅ Business Value
- Helps management monitor call center performance through real-time insights.
- Supports identification of high call volume periods for better resource planning.
- Enables evaluation of agent performance to improve service quality.
- Assists in identifying common customer issues to enhance support strategies.
- Facilitates data-driven decision making to improve overall customer experience.

## 🧠 Skills Demonstrated
- Data modeling using Star Schema and relationship management.
- Data transformation and cleaning using Power Query (M Code).
- Development of DAX measures for key performance metrics.
- Interactive dashboard design using slicers, filters, and tooltips.
- Use of Drill Down and Drill Through for detailed data analysis.
- Data visualization and storytelling using Power BI charts and matrix visuals.
- Analytical thinking to identify trends and performance insights.

## 📬 Conclusion
- The Call Center Dashboard provides a comprehensive view of call center operations and performance.
- It enables better monitoring of agent efficiency, response time, and customer satisfaction.
- The insights generated from the dashboard support data-driven decisions to improve service quality.
